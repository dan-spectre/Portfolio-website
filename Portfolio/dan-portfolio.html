Projects

IoT based Toilet Management System

This project is designed to provide automated toilet monitoring systems which can be implemented in various large institutions ranging from ational to medical to governmental buildings as well as implementing in public infrastructure. This project demonstrates the use of Proteus for simulation, coding with Arduino IDE and package importing and data input handling from sensors and translating them into usable and tangible output.

UiPath - RPA automation

RPA email automation streamlines repetitive email tasks using software robots. It can handle activities like reading, replying, forwarding, and organizing emails efficiently. Advanced bots can extract attachments, compile information, and even schedule events automatically.

Smart India Hackathon - AgriSense

Agrisense project is a Smart Poultry Coop management system which allows farmers to monitor the conditions of poultry farms, It's designed to be scalable and adaptable to various poultry farm sizes, aiding in agricultural and poultry productivity.

IoT Smart Waste Management System

Smart waste management systems using IoT optimize waste collection and disposal processes. Sensors in bins monitor fill levels and send real-time data to waste management teams, ensuring timely collection. IoT-enabled systems reduce operational costs, improve efficiency, and minimize environmental impact. These technologies contribute to cleaner and more sustainable urban environments.
